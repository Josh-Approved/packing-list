Packing List

A packing list that builds itself from the trip you're taking, then gets out of your way. Your data stays with you. No paywall.

I always forget the same few things on a trip — and over-pack everything else. So I built the checklist I actually wanted: tell it the kind of trip and how many days, and it lays out a sensible starting list. Rename anything, drag items between sections, split a list across the people you're packing for, and tick things off as you go.

There's no account and no pro tier — nothing to sign up for, nothing to upgrade. Your trips live in a local database on your phone. You can export them to a file and import them back yourself; that's the backup story for now. The code is on GitHub if you want to read exactly what it does.

What this app collects

Nothing about you. There's no account, no analytics, and no server that holds your data.

  • No analytics, no crash reporting, no install pings.
  • No accounts. Trips are stored locally and only leave the phone when you export them to a file yourself.
  • It checks for app updates, the way the App Store itself does — no identifiers, nothing about you. That's the only network use, and it's all in the source.
